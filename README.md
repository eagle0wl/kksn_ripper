usage:  
`> python kksn_ripper.py [.exe] [.nes]`  
`[.exe] is "KKSN DX.exe", "KKSN AC.exe" or "KKSN GOLD.exe".`  
`[.nes] is output file name.`  

example:  
`> python kksn_ripper.py "KKSN DX.exe" dx.nes`
